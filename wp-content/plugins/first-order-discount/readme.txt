Discount for the first order for Woocommerce.

== Description ==
First Order Discount allows you to create a discount for the first purchase of a registered user without any coupon.
You can choose type of discount: fixed discount or percent discount.

Added languages:

- english;
- russian;
- ukrainian.

**[You can upgrade your discount system with Advanced Discount plugin](https://codecanyon.net/item/advanced-discount-for-woocommerce/17790707)**

== Installation ==
You can use the built in installer and upgrader, or you can install the plugin
manually.

From WordPress dashboard:

1. Visit 'Plugins > Add New'.
2. Search for 'First Order Discount'.
3. Activate 'First Order Discount' from your Plugins page.

From WordPress.org

1. Download 'First Order Discount' zip file.
2. Upload the 'First Order Discount' directory to your '/wp-content/plugins/' directory.
3. Activate 'First Order Discount' Pack from your Plugins page.

Plugin settings:

1. Visit First Order Discount settings page (WooCommerce - First Order Discount)
2. Choose fixed, percent discount or disable discount.
